## Sobre o Bumba meu pão
O Bumba meu pão é um sistema para uma padaria que visa a responsividade. Com ele você pode gerenciar os produtos, categorias, clientes, pedidos, etc...
> [!IMPORTANT]
> Aplicativo da padaria Bumba meu pão.
